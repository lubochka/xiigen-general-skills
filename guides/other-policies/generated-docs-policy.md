# Generated Docs Policy

Generated docs, wiki exports, reports, and derived files are source material, not reusable guidance by default.

## Rules
1. Mark generated files as generated.
2. Keep canonical docs separate from generated docs.
3. Exclude raw generated corpora that contain private paths or internal references.
4. Extract reusable guidance only after human review.

## Gates
Generated docs are sanitized, classified, and linked to their source before use.

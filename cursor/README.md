# XIIGen Skills For Cursor Rules

This directory provides Cursor `.mdc` rules generated from the canonical
universal skills.

## Install From A Clone

Copy:

```text
cursor/rules/*.mdc
```

into the target project:

```text
.cursor/rules/
```

Cursor users often search for "Cursor rules", "Cursor workflows" and "AI coding
agent rules", so this pack uses Cursor-native `.mdc` files rather than expecting
Cursor users to discover `SKILL.md` folders.

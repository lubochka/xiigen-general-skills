# Source Boundary

This repository extracts and parameterizes guidance from source projects. It does not publish source projects, internal evidence, generated corpora, private prompts, copied worktrees, or raw command policies.

## Allowed

- Reusable project-independent agent rules.
- Parameterized architecture templates.
- Stack-skill factory guides based on target-project discovery.
- Sanitized examples.

## Not Allowed

- Raw `AGENTS.md` / `CLAUDE.md` copies as universal policy.
- Secrets, private paths, internal domains, raw evidence logs.
- Training data, model artifacts, experiments, generated bundles.
- Copied worktrees or private prompts.

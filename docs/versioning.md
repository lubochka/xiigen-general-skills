# Versioning

Current version: `0.1.0-oss-rc1`.

`0.1.0-oss-rc1` means the repository has open-source governance files and
sanitized reusable content, but public release still depends on the gates in
`docs/release-gates.md`.

## Policy

- Increment patch for wording and example changes.
- Increment minor for new skills, guides, mapping entries, or adapters.
- Increment major for breaking format changes to `adaptation-map.yaml`.

## Provenance

Every source-derived artifact records source family, transformation type, sanitizer status, and license status in the source coverage or mapping entry.

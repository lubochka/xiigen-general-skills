# XIIGen Smart Skills Pack for VS Code and Open VSX

This is a minimal extension wrapper scaffold for distributing XIIGen AI agent
skills through VS Code Marketplace or Open VSX.

It is not published yet. Review marketplace requirements, publisher identity,
icons, screenshots, extension signing, and support policy before publishing.

## Commands

- `XIIGen: Add Planning Skills`
- `XIIGen: Add Cursor Rules`
- `XIIGen: Open Skill Pack README`

## Safety

The extension copies reusable skills and rules into the current workspace. It
does not run project commands, upload files, change repository visibility, or
publish packages.

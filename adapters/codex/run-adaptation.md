# Codex Adaptation

Use `adaptation/ADAPTATION-RUNBOOK.md` and `adaptation/adaptation-map.yaml` as canonical input.

## Output Layout

- `AGENTS.md` for durable repository guidance.
- `.agents/skills/<skill-name>/SKILL.md` for reusable workflows.

## Rules

1. Keep startup guidance short.
2. Put long procedures into skills.
3. Every rendered skill or rule must reference the map entry id and canonical version.

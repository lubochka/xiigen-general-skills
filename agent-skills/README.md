# Agent Skills-Compatible Pack

This directory contains Agent Skills-style folders generated from the canonical
universal skills in `skills/universal/`.

Each folder contains a `SKILL.md` file with:

- YAML frontmatter: `name` and `description`.
- Concise task instructions.
- Inputs, steps, gates, evidence, anti-patterns and related skills.

Use this pack when a tool supports folder-based AI agent skills. Keep target
project commands and architecture-specific contracts outside these reusable
skills.
